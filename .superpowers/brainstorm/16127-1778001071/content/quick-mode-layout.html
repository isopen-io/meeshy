Annotations

  • Losange jaune = transition entre deux clips. Tap = inspector (kind, durée, easing). Drag = ajuste duration.
  • Petit losange jaune sur clip = keyframe. Tap = sélection. Long-press + drag = déplacer dans le temps.
  • Losange indigo = "duration handle" du slide. Drag horizontal = change la durée totale.
  • Trait blanc + triangle = playhead. Drag = scrub.
  • Poignées blanches gauche/droite sur clip sélectionné = trim (start / duration). Cap automatique sur la durée intrinsèque du média (sauf si loop activé).
  • "+ TEXTE • +1 PISTE" = piste cachée disponible (texte n'a pas de clip dans cet état). Tap = déploie tout. Swipe up sur la zone timeline aussi.
  • "PRO ↗" = bascule en mode Pro plein écran. Rotation paysage déclenche aussi automatiquement.
  • Fond noir + nav top semi-trans garde toute l'attention sur le canvas, comme sur Stories.

Règles de visibilité par défaut (état compact)

  1. Toujours afficher la piste contenant le clip actuellement sélectionné.
  2. Sinon, afficher en priorité : VIDEO 1 (background) puis AUDIO 1.
  3. Max 3 pistes en compact. Au-delà → trigger "+N PISTES" devient visible.
  4. Aucun élément d'un type → la piste ne compte pas (pas de slot vide affiché).

Réponds dans le terminal : OK pour valider, ou décris les ajustements souhaités (taille canvas, position des contrôles, autres pistes par défaut, etc.).
