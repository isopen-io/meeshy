# ASSETS OBLIGATOIRES POUR L'APP STORE

## 📱 ICÔNE DE L'APPLICATION

### Tailles requises (App Icon dans Assets.xcassets)

L'App Icon doit être présent dans toutes les tailles suivantes :

```
iPhone:
- 1024x1024px (App Store, sans alpha, sans transparence)
- 180x180px (@3x, iPhone)
- 120x120px (@2x, iPhone)
- 87x87px (@3x, iPhone Settings)
- 80x80px (@2x, iPhone Spotlight)
- 60x60px (@2x, iPhone Spotlight)
- 58x58px (@2x, iPhone Settings)
- 40x40px (@2x, iPhone Spotlight)

iPad (si vous supportez iPad):
- 1024x1024px (App Store)
- 167x167px (@2x, iPad Pro)
- 152x152px (@2x, iPad)
- 76x76px (@1x, iPad)
```

### Spécifications de l'icône

- Format : PNG
- Espace colorimétrique : sRGB ou P3
- Pas de transparence
- Pas de canal alpha
- Coins carrés (iOS ajoutera automatiquement les coins arrondis)
- Design reconnaissable même à petite taille

## 🎬 VIDÉO DE PRÉVISUALISATION (OPTIONNEL MAIS RECOMMANDÉ)

### Spécifications
- Durée : 15-30 secondes
- Format : .mov, .mp4, .m4v
- Résolution : Correspondant aux tailles de captures d'écran
- Codec : H.264 ou HEVC
- Pas de son obligatoire, mais recommandé

### Tailles vidéo
- iPhone 6.7" : 886 x 1920 pixels ou 1080 x 1920 pixels
- iPhone 6.5" : 886 x 1920 pixels ou 1080 x 1920 pixels
- iPad Pro 12.9" : 1200 x 1600 pixels ou 1600 x 1200 pixels

## 🎨 CONSEILS POUR LES CAPTURES D'ÉCRAN

1. **Ordre stratégique** :
   - 1re capture : Fonctionnalité principale (conversation avec traduction)
   - 2e capture : Appel vidéo
   - 3e capture : Liste des conversations
   - 4e capture : Profil utilisateur
   - 5e capture : Paramètres/fonctionnalités supplémentaires

2. **Design** :
   - Utilisez un fond blanc ou de couleur unie
   - Ajoutez du texte expliquant la fonctionnalité
   - Gardez une cohérence visuelle entre les captures
   - Utilisez de vraies données (mais anonymisées)
   - Évitez les écrans vides

3. **Outils recommandés** :
   - Sketch ou Figma pour créer des mises en page
   - Screenshots.pro pour des mockups professionnels
   - Apple Design Resources pour les device frames
