## Web — Admin Panel Hardcoded Strings

### RankingTable.tsx (`components/admin/ranking/RankingTable.tsx`)

- `getTitle()` switch returns 5 hardcoded French strings
- Badge: `{rankings.length} résultats` — French unit label
- Retry button: `Réessayer`
- Empty state: `Aucun résultat trouvé`
- Component has no `'use client'` directive (needed for `useI18n`)

### ScanLogTable.tsx (`components/admin/agent/ScanLogTable.tsx`)

- `formatTimeAgo()` — `'maintenant'`, `'min'` suffix, `'h'` suffix, `'j'` suffix (French day)
- Filter buttons — mixed French/English: `'Tous'` (French), `'Sent'`, `'Skip'`, `'Error'` (English)
- `TRIGGER_LABELS` — `'Manuel'` is French; others English
- `'Scan Logs'` title — English but not localized
- `'Aucun scan log'` empty state — French

### AgentTopicRegexTester.tsx (`components/admin/agent/AgentTopicRegexTester.tsx`)

- Section title: `'Tester regex contre texte sample'` — French
- Textarea placeholder: French long string
- Button: `'Tester'` — French
- No-patterns message: `'Aucun pattern défini pour ce topic.'`
- Match count: `` `${count} match${count !== 1 ? 'es' : ''}` `` — English pluralization, not i18n
- Error fallback: `'regex invalide'` — French

### MarkdownViewer.tsx (`components/markdown/MarkdownViewer.tsx`)

- `setErrorMessage('Impossible de charger le fichier')` — French, stored in state
- `title="Supprimer ce fichier Markdown"` — French delete button title
- `title={showRaw ? 'Vue formatée' : 'Vue brute'}` — French view toggle
- `title="Ouvrir en plein écran"` — French fullscreen button
- `title="Télécharger"` — French download anchor
