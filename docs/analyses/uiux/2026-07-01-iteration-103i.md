### 6 `.font(.system(size:))` gardés FIGÉS (commentés)
- État-vide `photo` 48pt — décoratif ≥40pt (doctrine 74i/86i) + `.accessibilityHidden(true)`.
- Chrome `xmark.circle.fill` 28pt — cadre tap icône + padding défaut ≈60pt (doctrine 82i).
- Chrome glyphe save 18pt — cadre tap fixe 40×40 (doctrine 82i).
- Contrôle `play.fill` 22pt / `arrow.down.to.line` 20pt / label taille 10pt — contenus d'un cercle
  de contrôle de taille fixe 56/64pt (scaler déborderait le cercle).

### VoiceOver — 8 traits déclaratifs ajoutés
- `.accessibilityLabel(common.close)` sur le bouton de fermeture (clé SSOT existante).
- `.accessibilityLabel(media.saveToPhotos)` + `.accessibilityValue(saveStateAccessibilityValue)` sur
  le bouton d'enregistrement (état saving/saved/failed annoncé, glyphe décoratif).
- `.accessibilityLabel(playOrDownloadAccessibilityLabel)` dynamique sur le bouton central vidéo
  (`media.playVideo` existante / `media.downloadVideo` / `common.downloading` selon `availability`).
- `.accessibilityElement(children: .combine)` sur la rangée auteur (avatar + nom + date → 1 lecture).
- `.accessibilityHidden(true)` sur les 2 glyphes décoratifs (état-vide `photo`, type média).
