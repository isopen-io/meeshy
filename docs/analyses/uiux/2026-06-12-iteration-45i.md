## iOS Findings

### Critical : Épuration — 11 structs legacy mortes (zéro usage vérifié)
L'audit d'usage (grep symbole par symbole sur Meeshy/MeeshyTests/MeeshyWidgets/extensions)
prouve que les wrappers « Legacy Support » ne sont référencés nulle part :
- `ConversationListHelpers.swift` — `SemanticColors` (avec sa `vibrantPalette` de 20 hex
  hors charte dont `4ECDC4`/`E91E63`/`9B59B6`), `ColorfulConversationRow`, `CommunityCard`,

  (hardcode `4ECDC4`)
- `ConversationHelperViews.swift` — `ConversationOptionButton`, `AttachOptionButton`,
  `MessageBubble` (hardcode `4ECDC4`), `ColorfulMessageBubble`

Plutôt que de re-peindre du code mort aux couleurs de la charte, on le supprime (logique
d'épuration de la routine). La suppression élimine d'un coup 3 occurrences `4ECDC4` et la
palette « arc-en-ciel » legacy de 20 couleurs.

### Critical : Ancienne palette en fallback UI (surfaces vivantes)
Règle charte : `4ECDC4` n'est légitime QUE dans l'algorithme `DynamicColorGenerator`
(couleur de type « group ») ; en fallback UI codé en dur, c'est une violation.
- `ThreadView.swift:22` — accent du thread : `parentMessage.senderColor ?? "4ECDC4"` →
  fallback `MeeshyColors.brandPrimaryHex`
- `ThreadView.swift:151,160` — rangées de réponses : `message.senderColor ?? "4ECDC4"` →
  fallback sur l'accent du thread (pattern identique à `ReplyThreadOverlay:214,276,283`
  qui fait `senderColor ?? accentColor`)
- `ConversationView.swift:712` — `MessageDetailSheet(contactColor: conversation?.accentColor
  ?? "#FF2E63")` → `?? MeeshyColors.brandPrimaryHex`
- `StoryViewerView.swift:519` — `StoryViewersSheet(accentColor: Color(hex: "4ECDC4"))` →
  accent de l'auteur du groupe courant `currentGroup?.avatarColor ?? brandPrimaryHex`
  (cohérent avec la ligne 1015 du même fichier qui fait déjà ce fallback) ; au passage la
  ligne 1015 inline `"6366F1"` → `MeeshyColors.brandPrimaryHex`
- `StoryTrayView.swift:98` — groupe de prévisualisation : `avatarColor: "FF2E63"` →
  `brandPrimaryHex`. **Bonus i18n** : `username: "Aperçu"` (ligne 97) est du français brut

- `StoryViewModel.swift:1419,1424` — `thumbnailColor: "4ECDC4"` sur les médias feed →
