## iOS — Dynamic Type : Dossier Contacts ✅ TRAITÉ

### Sévérité HAUTE

Le dossier `Features/Contacts/` n'avait reçu aucune migration Dynamic Type dans les itérations
1–4. Toutes les vues utilisaient `.font(.system(size: X))` avec des tailles fixes.

| File | Occurrences | Tailles observées |
|------|------------|------------------|
| `ContactsListTab.swift` | 10 | 11–48pt |
| `DiscoverTab.swift` | 14 | 11–32pt |
| `BlockedTab.swift` | 5 | 12–48pt |
| `RequestsTab.swift` | 12 | 11–48pt |
| `ContactsHubView.swift` | 3 | 10–13pt |

Mapping sémantique appliqué :
- 10–11pt → `.caption2`
- 12pt → `.caption`
- 13pt → `.footnote`
- 14–15pt → `.subheadline`
- 16pt → `.callout`
- 32–48pt (icônes état vide) → conservés `.system(size: X)` (décoratifs)
