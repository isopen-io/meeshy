# Analyse UI/UX — Itération 63w (web)

## Surface auditée
`apps/web/app/(connected)/me/page.tsx` — page **profil `/me`** (« Mon profil »).
Destination de navigation **primaire** : hero (avatar/bannière/nom/bio), stats (conversations/messages/contacts),
langues parlées, raccourcis (liens/contacts/notifs/paramètres), modales édition profil + déconnexion.

## Constat (rupture Prisme)
Le fichier importe `useI18n('settings')` et localise correctement ses **toasts** (`t('v2me.logoutSuccess')`,
`profileUpdated`, etc.) ainsi que le bouton `editProfile`. MAIS **28 chaînes user-facing restaient codées
en dur en français**, affichées telles quelles dans TOUTES les langues (en/es/pt) — un anglophone ou
lusophone voyait « Modifier le profil », « Se déconnecter ? », « Natif/Courant/Apprentissage », etc.
Surface d'entrée → rupture Prisme à fort impact.

Détail (28) : modale édition (8), modale déconnexion (5, dont `cancel` partagé), chrome (titre layout +
état erreur, 3), stats (aria + 3 labels, 4), langues (aria/titre + 3 niveaux, 5), raccourcis (aria + 4 labels
+ 2 boutons, 7) — avec recouvrement de clés partagées (`cancel`, `logout`, `myLanguages`).
+ 1 anti-pattern `t('title') || 'Paramètres'` (dead-code, cf. impl. `use-i18n.ts`).

## Correctif appliqué
- `useI18n('settings')` câblé dans `EditProfileModal`, `LogoutConfirmModal`, `ProfileShell`.
- 32 appels `t('v2me.*', '<EN>')` (signature fallback native, anti-flash, leçon 50w).
- Anti-pattern corrigé → `t('title', 'Settings')`.
- Bloc `settings.v2me` étendu de 5 → **33 clés ×4 locales** (parité vérifiée, JSON valide).
- `Pro` (tier produit) laissé non-localisé — marque universelle, cohérent avec l'existant.

## Vérifications
- Parité locale ×4 : ensembles de clés identiques (diff `en^l = ∅`).
- Grep FR résiduel sur le fichier = 0 (hors `Pro`).
- Aucun test n'asserte ces chaînes.
- `node_modules` absent (conteneur frais) → `tsc` non concluant localement ; CI = source de vérité.
  Aucune erreur `tsc` ne référençait `me/page` (seules les résolutions de modules manquantes
  du conteneur frais apparaissaient).
