## Context

Iterations 1–5 covered: MeeshyColors token migration (ReportUserView, FloatingCallPillView,
SettingsView, UniversalComposerBar send button, LanguagePickerSheet, MiniAudioPlayerBar,
ConversationView teal, DeleteAccountView, BubbleMetaBadges, ConversationInfoSheet, BubbleFooter,

## iOS — Accessibility

### HIGH

**`FeedPostCard+Media.swift` — missing `.accessibilityLabel` on tappable media cells (15 instances)**

All `galleryImageView().contentShape(Rectangle()).onTapGesture { openFullscreen(...) }` patterns
lack `.accessibilityLabel` and `.accessibilityAddTraits(.isButton)`. VoiceOver cannot identify
these interactive images.
