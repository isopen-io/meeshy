### MODERATE

**I6 ‚Äî ConversationDashboardView: ContentTypeStat uses hex String instead of Color**
- File: `apps/ios/Meeshy/Features/Main/Components/ConversationDashboardView.swift`
- Lines: 1082‚Äì1117 (struct definition + creation sites) + 778, 790 (usage sites)
