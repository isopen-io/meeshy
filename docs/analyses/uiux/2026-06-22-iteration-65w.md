## Classe d'anti-pattern ciblée
`t('key') || 'fallback'` — `use-i18n.ts` (`return fallback || key`) renvoie la **clé
brute** (truthy) pendant le load OU si la clé est absente ⇒ (1) le secours `||` est
**dead-code** jamais atteint et (2) la **clé brute flashe**. Correctif : signature
**native** `t('key', 'English')` (anti-flash, leçon 50w ; `fallbackLocale='en'`).

## Constat — double sévérité
`ConversationSettingsModal.tsx` (`useI18n('conversations')`) contenait **30 occurrences**
de `t(k) || 'FR'`. Deux niveaux de gravité :

1. **24 occ. = dead-code + flash** : clés présentes ×4 locales (`title`,
   `conversation`, `organization`, `pin`/`mute`/`archive` (namespace
   `conversationHeader`), `pinDescription`, `muteDescription`, `archiveDescription`,
   `customization`, `reaction`, `reactionTooltip`, `customReactionPlaceholder`,
   `savePreferences`, `security`, `encryptionMode`, `noEncryption`, `e2ee`, `hybrid`,
   `serverEncryption`, `e2eeDescription`, `hybridDescription`, `serverDescription`,
   `noEncryptionDescription`).
2. **6 occ. = VRAI bug i18n** : clés présentes **uniquement en `fr`**, ABSENTES de
   `en`/`es`/`pt` → tout utilisateur non-francophone voyait la **clé brute**
   (`conversationDetails.activity`, …) affichée littéralement dans l'UI (rupture
   Prisme + chaîne technique visible) :
   `activity`, `viewAllParticipants`, `mediaAndAppearance`, `changeBanner`,
   `uploadBanner`, `directConversationWarning`.

## Correctif
- **Code (1 fichier)** : 30 occ. `t(k) || 'FR'` → `t(k, 'English')`. Secours alignés
  sur la **valeur EN exacte** du locale pour les 24 clés présentes (les anciens
  secours FR divergeaient — ex. `title` codé `'Paramètres'` vs locale EN
  `'Conversation details'`). Pour les 6 clés ajoutées, secours = la nouvelle valeur EN.
- **Locales (en/es/pt, 3 fichiers, +6 clés chacun)** : ajout des 6 clés manquantes
  sous `conversations.conversationDetails.*` (déjà présentes en `fr`) → parité ×4
  rétablie. `fr` **non modifié**.
  - `activity` : Activity / Actividad / Atividade
  - `viewAllParticipants` : View all participants / Ver todos los participantes / Ver todos os participantes
  - `mediaAndAppearance` : Media & Appearance / Multimedia y apariencia / Mídia e aparência
  - `changeBanner` : Change banner / Cambiar banner / Alterar banner
  - `uploadBanner` : Add a banner / Añadir un banner / Adicionar um banner
  - `directConversationWarning` : A direct conversation should normally have only 2 participants. / Una conversación directa normalmente debería tener solo 2 participantes. / Uma conversa direta normalmente deve ter apenas 2 participantes.
