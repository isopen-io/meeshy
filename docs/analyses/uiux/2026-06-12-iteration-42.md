Inventaire : SettingsView ~45 occurrences (~30 hors charte), NotificationSettingsView ~48 (~20 hors
charte), DataExportView reliquat. Palette sauvage : `9B59B6`/`A855F7` (violets), `E91E63`/`EF4444`/
`FF6B6B` (rouges), `3498DB`/`45B7D1`/`3B82F6` (bleus), `4ECDC4` (teal), `2ECC71`/`27AE60`/`4ADE80`

violets→`brandDeep` (indigo700) ; rouges→`error` ; bleus→`info` ; teal→`indigo300` ;
verts→`success` ; oranges→`warning` ; `6366F1`→`brandPrimary`. Constantes hex `errorHex`/`infoHex`
ajoutées à `MeeshyColors` pour les paramètres `color: String`.

**Conservés intentionnellement** : OnboardingView (gradients décoratifs pagés — thème par page) ;
MessageComposer `FF2E63`/`08D9D6` (animations send/focus distinctives).
