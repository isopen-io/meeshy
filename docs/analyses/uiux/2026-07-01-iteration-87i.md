## iOS Findings

### Dynamic Type — `.font(.system(size:))` → `MeeshyFont.relative(...)` (9/9 sites)
Tous les sites vivent dans des conteneurs **flexibles** (capsules avec padding, HStack) — aucun
frame de dimension fixe → **tous peuvent scaler** (contrairement à AboutView 86i où `fieldIcon`
était dans un badge fixe). Migration mécanique préservant `weight` :

| Ligne | Rôle | Avant | Après |
|---|---|---|---|
| 23 | glyphe chip effet | `.system(size: 13)` | `MeeshyFont.relative(13)` |
| 25 | libellé chip effet | `.system(size: 12, weight: .medium)` | `MeeshyFont.relative(12, weight: .medium)` |
| 54 | titre « Effets du message » | `.system(size: 17, weight: .bold)` | `MeeshyFont.relative(17, weight: .bold)` |
| 57 | bouton OK | `.system(size: 15, weight: .semibold)` | `MeeshyFont.relative(15, weight: .semibold)` |
| 108 | titre de section | `.system(size: 13, weight: .semibold)` | `MeeshyFont.relative(13, weight: .semibold)` |
| 126 | titre durée éphémère | `.system(size: 13, weight: .semibold)` | `MeeshyFont.relative(13, weight: .semibold)` |
| 135 | pilule de durée | `.system(size: 12, weight: .medium)` | `MeeshyFont.relative(12, weight: .medium)` |
| 161 | compteur effets actifs | `.system(size: 12)` | `MeeshyFont.relative(12)` |
| 168 | bouton « Tout effacer » | `.system(size: 12, weight: .medium)` | `MeeshyFont.relative(12, weight: .medium)` |

### Palette — rouge destructif `.red` → `MeeshyColors.error` (SSOT)
Ligne 169 : `.foregroundColor(.red.opacity(0.8))` → `.foregroundColor(MeeshyColors.error.opacity(0.8))`.
Le bouton « Tout effacer » réinitialise tous les effets = action destructive. Le rouge système
(`Color.red`, plus vif/saturé) ne s'adapte pas à la charte ; `MeeshyColors.error` (#F87171) est le
token destructif unique — cohérence directe avec 78i (mêmes boutons destructifs des media viewers).

## Périmètre délibérément exclu (ne pas re-flagger)
- **Aucun glyphe figé** ici (contrairement à 86i) : tous les conteneurs sont flexibles.
- **Couleurs d'accent de sélection** `Color(hex: accentColor)` : accent déterministe passé en
  paramètre (accent de conversation) — SSOT existant, pas un littéral à tokeniser.
- **`Color.gray.opacity(0.1)`** (fond de chip inactif) et `.secondary` (texte inactif) : rôles
  neutres système standard, hors-scope palette.
- i18n : déjà complet (aucun littéral figé restant).

## Anti-repetition check
Vérifié contre le § History et les pointeurs autoritaires : **aucune** itération n'a touché
`EffectsPickerView`. Distinct de l'emoji-picker (#1154, `EmojiPickerSheet` VoiceOver labels) et

71i 2FA, 72i FeedComments, #1191/#1188 audio player, #1189 sessions). Le swap `.red`→`error`
prolonge la doctrine 78i (rouges destructifs → token) sur un site non couvert par 78i.
