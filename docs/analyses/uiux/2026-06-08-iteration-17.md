## Web Issues

### HIGH — `aria-label` hardcoded French in TextLightbox (3 instances)

**File:** `apps/web/components/text/TextLightbox.tsx`

TextLightbox already has `useI18n('common')`. Need to add `useI18n('viewers')` for text-specific keys.
All keys already exist in `viewers.json` across all 4 locales.

| Line | Before | After |
|------|--------|-------|
| 234 | `aria-label={wordWrap ? 'Désactiver le retour à la ligne' : 'Activer le retour à la ligne'}` | `aria-label={wordWrap ? tViewers('text.disableWordWrap') : tViewers('text.enableWordWrap')}` |
| 249 | `aria-label="Copier le contenu"` | `aria-label={tViewers('text.copy')}` |
| 273 | `aria-label="Télécharger le fichier"` | `aria-label={tViewers('text.download')}` |
