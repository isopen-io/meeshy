## Problème traité — `ReelPlayer` 100 % FR figé (rupture Prisme + a11y)
`components/feed/ReelPlayer.tsx` (lecteur immersif plein écran reel/short) n'avait
**aucun** hook i18n : 13 chaînes user-facing **françaises figées en TOUTES langues**
— dont la quasi-totalité sont des `aria-label`/`sr-only`, donc invisibles à l'œil
mais lues par les lecteurs d'écran dans la mauvaise langue. Surface de **contenu**
partagée (montée par `app/feed/reels`, `app/reel/[postId]`, `ReelsFeedScreen`).

| Chaîne FR figée | Correctif |
|-----------------|-----------|
| `aria-label={`Reel ${i+1} sur ${total}`}` (section) | `t('player.position', { index, total })` |
| `<h1 sr-only>Reel de {name}` | `t('player.reelBy', { name })` (variable `reelBy`) |
| `aria-label={video.alt ?? `Reel de ${name}`}` | `video.alt ?? reelBy` |
| `aria-label="Lecture"` (overlay play) | `t('player.play', 'Play')` |
| `alt={image.alt ?? `Reel de ${name}`}` | `image.alt ?? reelBy` |
| `aria-label="Fermer"` | `t('player.close', 'Close')` |
| `aria-label="Reel précédent"` | `t('player.previous', 'Previous reel')` |
| `aria-label="Reel suivant"` | `t('player.next', 'Next reel')` |
| `aria-label={muted ? 'Activer le son' : 'Couper le son'}` | `t('player.unmute'/'player.mute')` |
| RailButton `J’aime`/`Commenter`/`Partager`/`Enregistrer` | `t('player.{like,comment,share,save}')` |

Hook `useI18n('reel')` ajouté ; le namespace `reel.json` existait déjà (créé en
53wb pour la page deep-link) — extension par un **sous-bloc `player`** (12 clés)
plutôt qu'un nouveau namespace : sémantiquement cohérent (même domaine reel),
zéro fichier supplémentaire.

## Parité iOS (étape 5)
`apps/ios/.../ReelsPlayerView.swift` + `FeedCommentsSheet.swift` localisent
**toutes** leurs a11y via `accessibilityLabel(String(localized:…))`. Le web était
le seul frontend à laisser ces labels en FR brut → 57w aligne le web sur le
pattern iOS (a11y localisée, labels d'action standard Like/Comment/Share/Save).

## Décisions
- Sous-bloc `reel.player` (12 clés ×4 locales) plutôt qu'un namespace neuf —
  même domaine, self-documenting, pas de nouveau fichier runtime.
- Fallbacks EN en 2e arg de `t()` pour les 10 chaînes simples (anti-flash, leçon
  50w). Les 2 clés à paramètre (`position {index,total}`, `reelBy {name}`)
  n'acceptent pas de fallback string (signature `t()` = params OU fallback) ;
  elles existent à parité ×4 → aucun flash de valeur brute.
- `reelBy` factorisé en variable locale (3 usages : h1 sr-only, alt vidéo, alt
  image) — DRY, une seule interpolation.
- Diffs locale strictement additifs (les 11 clés existantes 53wb intactes).
