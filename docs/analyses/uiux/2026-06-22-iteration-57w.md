**Date** : 2026-06-22
**Périmètre** : application web uniquement (`apps/web`)
**Base** : `main` HEAD post-merge iter-56w (#771) — commit `bf731cb`

## Contexte

Reprise du différé **« `ReelPlayer` + surface feed non internationalisés »** noté dès 53w.
Vérification : `components/feed/ReelPlayer.tsx` (lecteur reel immersif plein écran)
n'avait **aucun hook i18n** — toutes ses chaînes user-facing étaient en **français figé**,
affichées en TOUTES langues. C'est une rupture du Prisme Linguistique UI sur une surface
partagée : le composant est monté à la fois par la page deep-link `/reel/[postId]`
(jumelle déjà i18n en 53wb) **et** par `ReelsFeedScreen`.

## Constat (audit ciblé `ReelPlayer.tsx`)

## Anti-duplication

- Aucune analyse antérieure ne couvrait `ReelPlayer.tsx` (seule la page conteneur
  `/reel/[postId]` l'a été en 53wb — strings de toasts/états/loading, périmètre disjoint).
- Le namespace `reel` existe déjà (créé 53wb) → **réutilisé** (pas de nouveau fichier).
  Les nouvelles clés sont regroupées sous un objet `player` pour éviter toute collision
  avec les clés deep-link existantes (`linkCopied`, `loading`, `unavailableTitle`, …).

## ✅ CORRIGÉ ET COMPLET — NE PLUS RE-FLAGGER

`components/feed/ReelPlayer.tsx` est **entièrement internationalisé** (12 clés
`reel.player.*` × 4 locales en/fr/es/pt) :
- hook `useI18n('reel')` câblé ;
- clés simples avec fallback EN en 2e arg (anti-flash, leçon 50w) ;
- 2 clés paramétrées (`player.position {current}/{total}`, `player.byAuthor {name}`)
  sans fallback string (signature `t()` exclusive — params XOR fallback) ; parité ×4
  garantit zéro flash de clé brute (et ces 2 cibles sont sr-only/aria, invisibles).

**Reste du cluster feed (différé 58w+, NON couvert ici)** : `PostsFeedScreen.tsx`
(`aria-label` FR : `Type de fil`, `Stories publiques`, `Humeurs`, `Composer une publication`,
`Enregistrer un post audio`, `Publications`, `Chargement de plus de publications` ;
`title="Feed"`) et `ReelsFeedScreen.tsx` (`title="Reels"`, `DashboardLayout`). Bornés,
à traiter en lot dédié.
