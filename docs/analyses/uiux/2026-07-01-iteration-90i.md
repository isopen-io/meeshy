## Diagnostic

`MagicLinkView` (écran modal de connexion par lien magique, 2 étapes `emailInput`/`waiting`)
utilisait **17** `.font(.system(size:))` à taille **figée** — le texte ne suivait donc pas
le réglage Dynamic Type de l'utilisateur. Les tailles étaient déjà exprimées via les
constantes sémantiques `MeeshyFont.{titleSize,headlineSize,subheadSize,bodySize}` mais
enveloppées dans `.system(size:)` (point fixe, non scalable) — rupture de la règle a11y
CLAUDE.md « never fixed font sizes for body text ».

### Classement des 17 sites

**14 sites texte-de-lecture + glyphe inline apparié → `MeeshyFont.relative(...)`** (scale en
tandem, weight/design préservés) :
- Titre principal de la toolbar (`auth.magiclink.title`)
- Étape email : titre + sous-titre + message d'erreur + libellé du bouton d'envoi
- Étape attente : titre « Lien envoyé ! » + sous-titre + email affiché + message d'expiration
  + instructions + **compteur d'expiration** (`.monospacedDigit()` préservé)
- Bouton « Renvoyer » : glyphe `arrow.clockwise` (inline, apparié au texte) + libellé HStack
- Bouton « Annuler »

**3 sites gardés FIXES (justifiés & commentés)** :
- **Glyphe `xmark` de la barre de navigation** (`headlineSize`/17) — chrome de navigation,
  cadre de tap géré par la toolbar → taille stable (doctrine chrome 82i/87i).
- **Héros `wand.and.stars` 56pt** (étape email) — décoratif ≥40pt.
- **Héros `envelope.open.fill` 48pt** (étape attente) — décoratif ≥40pt.

## Corrections appliquées

1. **Dynamic Type** : 14 swaps mécaniques `.font(.system(size: X, weight: W))` →
   `.font(MeeshyFont.relative(X, weight: W))` (weight préservé ; `.monospacedDigit()`
   conservé sur le compteur d'expiration).
2. **VoiceOver** :
   - `.accessibilityLabel("common.close")` (clé SSOT réutilisée, 0 clé neuve) sur le bouton
     de fermeture `xmark` qui n'avait aucun label — annonçait « bouton » vide.
   - `.accessibilityHidden(true)` sur les 2 héros décoratifs (`wand.and.stars`,
     `envelope.open.fill`) — bruit VoiceOver éliminé.
3. **3 sites figés commentés** avec justification d'exception (chrome / décoratif ≥40pt).
