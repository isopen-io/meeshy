### MEDIUM — DetailTab.color: String hex values

**File:** `apps/ios/Meeshy/Features/Main/Components/MessageDetailSheet.swift`
Lines 44–57

`DetailTab.color` returns hex strings used for tab indicator pills. Most don't map cleanly to `MeeshyColors` (custom palette for functional differentiation). **Defer to iteration 15** — not all have semantic equivalents, and changing would require changing the `color: String` return type to `Color` across usage sites.

## Strategy

Due to volume (106 font instances), use systematic sed-like replacements grouped by size:
1. Fix French defaultValues first (small, safe)
2. Migrate non-monospaced sizes by pattern
3. Migrate monospaced sizes separately (different syntax)
4. Keep size 28 and 48 as-is (display numerals / decorative)
