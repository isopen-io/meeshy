## Corrections appliquées

1. **Dynamic Type — 7/9 sites** `.font(.system(size: N[, weight:]))` → `MeeshyFont.relative(N[, weight:])` (weight préservé) :
   - `TextEditor` du corps (17), compteur de caractères (12 medium), bouton « Publier » (16 semibold),
   - libellé « Langue du contenu » (15), valeur langue sélectionnée (15 medium), « Auto » (15),
   - chevron trailing du sélecteur de langue (12 semibold, glyphe inline apparié au libellé scalable).
2. **2 glyphes gardés FIXES & commentés** (dans une vignette média rigide 64×64 —
   doctrine : un glyphe épinglé dans un cadre fixe crève sa frame s'il scale) :
   - bouton **retirer/restaurer** (18) → **+ `.accessibilityLabel`** (lacune comblée : action « Retirer le média » / « Restaurer le média »).
   - icône média placeholder (22) → **+ `.accessibilityHidden(true)`** (décoratif).
3. **VoiceOver** : `.accessibilityLabel` sur le `TextEditor` (« Contenu du post »)
   et sur le compteur (« %d caractères restants ») → lecture claire.
