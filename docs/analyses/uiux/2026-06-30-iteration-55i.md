## Contexte / point de départ
- `ConversationInfoSheet` (fiche conversation : header hero/direct, onglets Membres/Médias/
  Stats/Options, messages épinglés, section sécurité E2E, boutons d'action) était la **plus
  grosse surface jamais migrée** : **52 occurrences `.font(.system(size:))` toutes figées**
  (`MeeshyFont.relative` count = 0 avant cette itération), listée en tête des « différés grandes
  surfaces » par 54i.
- L'atome `MeeshyFont.relative(size, weight:design:)` mappe une taille point legacy vers le
  `Font.TextStyle` relatif le plus proche (`textStyle(for:)`) → la police **scale avec le réglage
  Dynamic Type** tout en préservant weight/design. Migration = swap mécanique :
  `.font(.system(size: 15, weight: .medium))` → `.font(MeeshyFont.relative(15, weight: .medium))`.
- Apple HIG : « NEVER use fixed font sizes for body text » (cf. `apps/ios/CLAUDE.md` §
  Accessibility).
